# Tic-Tac-Toe (Kevin Javier Ruiz Tillit) — entregable

Proyecto React listo para entregar y desplegar en GitHub Pages.

## Qué contiene
- Juego de Tic-Tac-Toe basado en el tutorial oficial de React.
- Opciones añadidas:
  - Cambiar tamaño del tablero (3 a 6).
  - Reemplazar X/O por imágenes (SVG) mediante toggle.
  - Temas y CSS en blanco / negro / rojo.
  - Easter egg con datos del alumno: **Kevin Javier Ruiz Tillit** (detalles abajo).
- Configurado para desplegar con `gh-pages`.

## Deploy rápido (GitHub Pages)
1. Reemplaza `"homepage"` en `package.json` por `https://<tu-usuario>.github.io/<tu-repo>`.
2. `git init` -> commit -> subir a GitHub como `<tu-repo>`.
3. Ejecuta:
```bash
npm install
npm run deploy
```

## Easter egg
Haz clic 7 veces rápido en el logo de la esquina superior izquierda (o presiona la tecla `k` 5 veces) para mostrar la tarjeta oculta con los datos del alumno.

## Nota
Este zip incluye todo el código fuente. Ejecuta `npm install` localmente antes de `npm start`.
